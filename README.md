# media-copytool
PowerShell-based, GUI-including script that not only copies your files, but also verifies them afterwards.

I recommend using the readme-file `README.rtf` (fill in * according to the version you are using) or using `Get-Help .\media_copytool.ps1 -detailed` in PowerShell - as of now, I still have to get used to GitHub's styling (and its limitations for a complete readme).
